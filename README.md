# podcast

Local-first, NotebookLM-style podcast generator: feed it documents, get a two-host
dialogue script you can edit, then a rendered audio episode — all on your own machine.

> Full documentation (installation, Strix Halo GPU setup, hardware verification
> checklist) lands with the MVP; see
> [docs/superpowers/specs/2026-07-10-podcast-generator-design.md](docs/superpowers/specs/2026-07-10-podcast-generator-design.md)
> and [docs/adr/](docs/adr/) for the design, and
> [docs/python_development_standards_v3.md](docs/python_development_standards_v3.md)
> for the engineering standards this project follows.
